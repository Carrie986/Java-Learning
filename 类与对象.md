## 类与对象：

- 基本数据类型： 整数型 浮点型 字符型 布尔型
    - 创建的变量只能表示单个数据
    - int n = 10;
- 像学生这样的数据体：
    - 我们需要自己定义复合型数据类型表示
    - 简称为类 ，类中可以定义多个不同的数据变量
    - 一个学生 包含 姓名 学分 年龄 各科成绩
- 在程序中表示现实世界的事物时，会将其数据化


- 类与对象：
    - 类： class 类别
    - 对象： object 具体的事物
    - PS:
        - 人 计算机 书 类别
        - 某个人 我正在使用的计算机 对象
    - 通过分析这些事物是否具有相同的属性、特征、行为、功能来划分类别 

- 类的定义： 
  - 格式： class 类名{ }
  - 内容： 
    - 属性： 表示这个类别都会有的数据信息  数据
      - 格式： 属性数据类型 属性变量名;
        - String name; int age; 
    - 方法： 表示这个类别都会有的功能/行为 功能
      - 格式：修饰词 返回值类型 方法名(参数列表){ 方法体 }
        - 1： public void 方法名(){ }
        - 2:  返回值与参数可以选择不用 
          - 返回值: 
            - void 表示没有返回值
            - 数据类型： 在方法结束的位置 需要返回一个对应的数据/变量
            - 在调用方法的式子上 可以获取返回值 
          - 参数： 在定义方法的时候 预设一组变量
            - 用于完成方法内部的逻辑编写
            - 调用方法时必须传入对应的数据 

- 创建对象： 自己定义的数据类型 创建变量，这种变量包含多个数据，叫做对象变量
  - 格式： 类名 对象变量名= new 类名();
    - ·PS ： `Student stu1 = new Student();`
  - 对象调用属性： 
    - 格式： 对象变量名.属性名 
      - stu1.name="张三";
  - 对象调用方法：
    - 格式： 对象变量名.方法名();
      - int t = stu1.getTotal();

```java
package com01.oop.zyf0107;

public class MathTools {

    public int abs(int n) {
        if (n < 0) {
            return -n;
        } else {
            return n;
        }
    }

    public static void main(String[] args) {
        MathTools mathTools = new MathTools();

        // 获取返回值 方法返回的结果会赋值给 n
        int n = mathTools.abs(-100);
        System.out.println(n);
    }

}
```

```java
package com01.oop.zyf0107;

import java.util.Scanner;

public class Student {
    // 属性： 姓名 英语 语文 数学
    String name;
    double eng;
    double math;
    double chinese;
    double total;


    // 计算总分
    public double getTotal() {
        total = eng + math + chinese;
        return total;
    }


}

class Main {

    public static void main(String[] args) {
        // 输入一个数字n 表示接下来有n名学生的信息
        // 输入n行数据 按照 姓名 英语 数学 语文的顺序排列
        // 输出 总分最高的前3名同学
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        // 根据n创建一个学生类型的数组
        Student[] stuList = new Student[n];

        for (int i = 0; i < n; i++) {
            // 读取一行数据
            String name = sc.next();
            double e = sc.nextDouble();
            double m = sc.nextDouble();
            double c = sc.nextDouble();
            // 创建一个具体的学生对象
            Student stu = new Student();
            // 将读取到的数据 存入对象中
            stu.name = name;
            stu.eng = e;
            stu.math = m;
            stu.chinese = c;
            // 计算总分
            stu.getTotal();
            // 将对象存入数组中
            stuList[i] = stu;
        }
        // 查找分数最高的前三个
        double max = stuList[0].total;
        int index = 0;
        for (int i = 0; i < n; i++) {
            if (max < stuList[i].total) {
                max = stuList[i].total;
                index = i;
            }
        }
        System.out.println("分数最高的同学是：" + stuList[index].name);
        // 查找分数最高的前三个
        double[] maxList = new double[3];
        int[] indexList = {0, 1, 2};
        for (int i = 0; i < 3; i++) {
            maxList[i] = stuList[i].total;
            indexList[i] = i;
        }
        for (int i = 0; i < 3; i++) {
            for (int j = 0; j < 3 - 1; j++) {
                if (maxList[j] > maxList[j + 1]) {
                    double temp = maxList[j];
                    maxList[j] = maxList[j + 1];
                    maxList[j + 1] = temp;

                    // 将下标也交换
                    int tempIndex = indexList[j];
                    indexList[j]=indexList[j+1];
                    indexList[j+1]=tempIndex;
                }
            }
        }
        for (int i = 0; i < 3; i++) {
            int si =  indexList[i];
            System.out.println(stuList[si].name+" "+stuList[si].total);
        }
        // 7 6 8
        //6 8 1 2 7 2 9 1 3 5 6 7 8
        for (int i = 3; i < n; i++) {
            Student stu = stuList[i];

            for (int k = 0; k < 3; k++) {
                for (int j = 0; j < 3 - 1; j++) {
                    if (maxList[j] > maxList[j + 1]) {
                        double temp = maxList[j];
                        maxList[j] = maxList[j + 1];
                        maxList[j + 1] = temp;

                        // 将下标也交换
                        int tempIndex = indexList[j];
                        indexList[j]=indexList[j+1];
                        indexList[j+1]=tempIndex;
                    }
                }
            }

            // stu的总分 需要与 maxList中的分数 从小到大比较
            for (int j = 0; j < maxList.length; j++) {
                if (stu.total > maxList[j]) {
                    maxList[j] = stu.total;
                    indexList[j] = i;
                    break;
                }
            }
        }

        for (int i = 0; i < 3; i++) {
          int si =  indexList[i];
            System.out.println(stuList[si].name+" "+stuList[si].total);
        }

        // 练习： 直接以stuList数组中的总分进行排序 输入打印前三个

    }

}
```
