# 事件监听器

## 监听器的作用
- 用来处理用户的交互行为（如：点击按钮、输入回车等）
- 在 Swing 中，按钮点击事件主要使用 `ActionListener `来监听
## 事件监听器接口
- Java 中的监听器本质是一个接口
- 按钮点击事件对应的接口：
```java
//用户点击按钮后，会自动调用 actionPerformed() 方法
public interface ActionListener {
    void actionPerformed(ActionEvent e);
}
```
## 监听器开发步骤
### 步骤一：创建监听器类
- 自定义一个类,实现 `ActionListener` 接口
```java
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class LoginAction implements ActionListener {

}
```
### 步骤二：重写核心方法
- 作用：当按钮被点击时，这个方法会自动执行
```java
public void actionPerformed(ActionEvent e) {

}
```
### 步骤三：在监听器中获取UI数据
- 用来接收外部传进来的输入框对象
```java
JTextField jt2;
JPasswordField jpf3;
```
### 步骤四：编写逻辑部分
```java
public void actionPerformed(ActionEvent e){
    System.out.println("按钮被点击了");

    String name = jt2.getText();
    String pwd = jpf3.getText();

    if (name.equals("YY") && pwd.equals("123456")) {
        System.out.println("登录成功！");
    }
    else {
        System.out.println("账号或者密码错误");
        jt2.setText("");
        jpf3.setText("");
    }
}
```
## 在UI中绑定监听器
- 作用：按钮点击时自动调用监听器方法
```java
LoginAction la = new LoginAction();
btn.addActionListener(la);
```
## UI与监听器数据传递
- 作用：把界面中的输入框对象传给监听器
```java
la.jt2 = jt;
la.jpf3 = jt1;
```
## 值传递与引用传递：
### 值传递：
- 复制值然后传递
- 如下图中的例子，a不变，b改变
```java
int a = 10;
int b = a;
b++;
```
### 引用传递：
- 传递对象的地址
- 两个变量指向一个对象
```java
User user1 = new User("张三");
User user2 = user1;
```
- 修改其中一个对象的变量，另一个也跟着变化
## 完整代码
- LoginUI:
```java
//导入界面包
import javax.swing.*;
import java.awt.*;

public class LoginUI {
    public void showUI () {

        //窗体开发
        JFrame jf = new JFrame();
        //标题，尺寸，关闭操作，布局，可视化
        jf.setTitle("登录界面");
        jf.setSize(300,200);
        jf.setLocationRelativeTo(null);
        jf.setDefaultCloseOperation(3);
        FlowLayout fl = new FlowLayout();
        jf.setLayout(fl);
        JLabel jl = new JLabel("账号：");
        jf.add(jl);
        JTextField jt = new JTextField(20);
        jf.add(jt);
        JLabel jl1 = new JLabel("密码：");
        jf.add(jl1);
        JPasswordField jt1 = new JPasswordField(20);
        jf.add(jt1);
        JButton btn = new JButton("登录");
        jf.add(btn);
        //创建监听类对象
        LoginAction la = new LoginAction();
        btn.addActionListener(la);
        la.jt2 = jt;
        la.jpf3 = jt1;

        jf.setVisible(true);

    }

    public static void main(String[] args) {
        LoginUI ui = new LoginUI();
        ui.showUI();
    }
}
```
- LoginAction:
```java
import javax.swing.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;
// 创建监听器
public class LoginAction implements ActionListener {
    JTextField jt2;
    JPasswordField jpf3;
    public void actionPerformed(ActionEvent e){
        System.out.println("按钮被点击了");
        String name = jt2.getText();
        System.out.println(name);
        String pwd = jpf3.getText();
        System.out.println(pwd);
        if (name.equals("YY")&&pwd.equals("123456")) {
            System.out.println("登录成功！");
        }
        else {
            System.out.println("账号或者密码错误");
            jt2.setText("");
            jpf3.setText("");
        }

    }
}
```
