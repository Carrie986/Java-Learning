# 界面开发

## 基础语法
- 类的定义：` public class 类名{}`
- 方法的创建：`public void 方法名(){}`
- 对象的创建：`类名 对象变量名=new 类名();`
- 对象调用方法：`对象变量名.方法名();`
## 界面的组成
- 可视化组件部分：窗体，按钮，输入框
- 元素，规则部分：尺寸，颜色，字体，布局
- 内容部分：文本，图片

## 界面开发包
- 包的作用：用来管理代码文件，在使用其他包中的类时，需要先导入包路径。
- 导入语法：`import 包名.*；` `import 包名.类名；`
- java.awt：早期的界面开发包，主要使用元素规则类；
	- Color 颜色类
	- Dimension 尺寸类
	- FlowLayout 流式布局类 （从左到右依次摆放）
- javax.swing：升级后的界面开发包
	- JFrame 窗体类
	- JButton 按钮类
	- JLabel 标签类
	- JTextField 文本输入框类
	- ImageIcon 图片类

## 开发步骤：
### 步骤一：搭建一个基础结构
- 创建一个类，在类中定义一个方法，用来开发界面，再到主函数中调用这个方法。
```java
package Com2026.ljy0611;

public class LoginUI {
    public void showUI () {
        
    }

    public static void main(String[] args) {
        LoginUI ui = new LoginUI();
        ui.showUI();
    }
}
```

### 步骤二：窗体开发
- 创建一个窗体，并设置窗体的相关属性
```java
package Com2026.ljy0611;
//导入界面包
import javax.swing.*;
import java.awt.*;

public class LoginUI {
    public void showUI () {

        //窗体开发
        JFrame jf = new JFrame();
        //标题，尺寸，关闭操作，布局，可视化
        jf.setTitle("登录界面");
        jf.setSize(300,200);
        jf.setLocationRelativeTo(null);
        jf.setDefaultCloseOperation(3);
        FlowLayout fl = new FlowLayout();
        jf.setLayout(fl);
        jf.setVisible(true);

    }

    public static void main(String[] args) {
        LoginUI ui = new LoginUI();
        ui.showUI();
    }
}
```

### 步骤三：添加组件
- 创建需要的组件对象，其中标签和按钮需要添加文本；输入框需要添加宽度。然后再按照流式布局的规则添加到窗体上。
```java
package Com2026.ljy0611;
//导入界面包
import javax.swing.*;
import java.awt.*;

public class LoginUI {
    public void showUI () {

        //窗体开发
        JFrame jf = new JFrame();
        //标题，尺寸，关闭操作，布局，可视化
        jf.setTitle("登录界面");
        jf.setSize(300,200);
        jf.setLocationRelativeTo(null);
        jf.setDefaultCloseOperation(3);
        FlowLayout fl = new FlowLayout();
        jf.setLayout(fl);
        JLabel jl = new JLabel("账号：");
        jf.add(jl);
        JTextField jt = new JTextField(20);
        jf.add(jt);
        JLabel jl1 = new JLabel("密码：");
        jf.add(jl1);
        JPasswordField jt1 = new JPasswordField(20);
        jf.add(jt1);
        JButton btn = new JButton("登录");
        jf.add(btn);

        jf.setVisible(true);

    }

    public static void main(String[] args) {
        LoginUI ui = new LoginUI();
        ui.showUI();
    }
}
```
