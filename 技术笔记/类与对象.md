# 类与对象

## 类与对象的基础概念

- 类：Class 类别
- 对象：Object 具体的事物
- 比如：类就是一个类别的意思，像人类，计算机类，学生类；
  - 对象就是这个类别具体的一个实例，比如张三，某一个具体的学生
- 类就是一个抽象的模板，包含一组相关的属性和行为。类可以描述对象的特征，对象是类的一个实例，在内存中是一个数据体

## 类的定义：组成与结构

- 格式：class 类名 {}

- 成员属性：用来描述这个类别都有的数据信息

  - 格式：数据类型 变量名；

- 成员方法：定义对象的行为，功能；比如学生会有吃饭，学习，等方法

  - 格式：public void 方法名(){}

- 代码样例：

- ``` java
  public class Student{
      //成员属性
      String name;
      int age;
      //成员方法
      pubic void study (){
          System.out.println(name+"正在学习");
      }
  }
  ```

## 对象的创建

- 格式：类名 对象变量名= new 类名（）；实例化类，包括声明，实例化和初始化三步

- 访问对象成员：

  - 访问属性：对象名.属性名=值；如stu.name=“张三”；
  - 调用方法：对象名.方法名（）；如stu.study();

- ``` java
  Student stu = new Student();
  ```
