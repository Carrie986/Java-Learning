# 类型转换

### 自动类型转换（隐式转换）

- 发生在取值范围更小的类型赋值给取值范围更大的类型时。
- 顺序（从小到大）：
byte→short→int & char→long→float→double
```java
// 从char到int的自动转换
char ch = 'A';
int charUnicode = ch; // 'A'的Unicode值是65
System.out.println("'A'的Unicode值: " + charUnicode); // 65
```
### 算术运算中的自动提升
- byte & short & char→int→long→float→double
- 不同类型一起运算时，向更大的类型提升：
```java
int a = 10;
double b = 3.5;
double sum = a + b;
```
- 方法参数传递也会自动提升：
```java
public static void printNumber(double x){
    System.out.println(x);
}
int a = 10;
printNumber(a); // 输出10.0
```
### 强制类型转换（显式转换）
- 将取值范围大的类型转换为取值范围小的类型时，需要强制转换，可能丢失数据。
- 强制类型转换语法：(目标类型) 值
```java
public class Test {
public static void main(String[] args) {

//double→int
double d = 9.99;
int i = (int) d; // 直接截断取整数部分，结果为9
System.out.println("int d" + i); // 9（精度丢失）

//long→byte
long l = 130L;
byte b = (byte) l; // 130超出byte范围-128~127，溢出
System.out.println("byte l" + b); // -126（数据溢出）
// 溢出原理解释：
// byte范围：-128 ~ 127（共256个数）
// 130 - 256 = -126

//float→int
float f = 2.6F;
int i2 = (int) f;
System.out.println("int f" + i2); // 2

// int→char
int unicode = 97;
char ch = (char) unicode;
System.out.println("Unicode 97对应的字符: " + ch); // 'a'

// 常见的实用转换
// int → String
int num = 100;
// 方法一：String.valueOf()
String str1 = String.valueOf(num);
// 方法二：拼接空字符串
String str2 = num + ""; 
System.out.println("int → String: " + str1);
System.out.println("int → String: " + str2);

// String → int :Integer.parseInt()
String str3 = "100";
int num3 = Integer.parseInt(str3);
System.out.println("String → int: " + num3);

// String → double：Double.parseDouble()
String str4 = "3.1415926";
double num4 = Double.parseDouble(str4);
System.out.println("String → double: " + num4);
}
}
```
### 类型转换规则表
| 转换方向                  | 是否强制转换 | 是否丢失数据   |
| ------------------------- | ------------ | -------------- |
| byte→short                | 否           | 否             |
| byte→char                 | 需要         | 可能（负数）   |
| short→int                 | 否           | 否             |
| int→long                  | 否           | 否             |
| int→float                 | 否           | 是（精度）     |
| long→float                | 否           | 是（精度）     |
| float→double              | 否           | 否             |
| double→int                | 需要         | 是             |
| char→int                  | 否           | 否             |
| int→char                  | 需要         | 可能（超范围） |
| boolean和其他类型互相转换 | 不能转换     | /              |
