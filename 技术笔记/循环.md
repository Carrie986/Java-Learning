## 翻转数字 字符串，比较数字/字符串的每一位 

```java

package oop.zyf1224;

import java.math.BigInteger;
import java.util.Scanner;

public class Main {
    // 2 12 20 21 22
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int x = sc.nextInt();
        int count=0;
        for (int i = 1; i <=n; i++) {
            // i=999  x=9
            int m=i; // 如何取到一个数的每一位 整除 和 取余
           while(m>0){
               int l = m%10;//l=9
               if(l==x){
                   count++;
               }
               m/=10;// 整数相除 999/10=99
           }
        }
        System.out.println(count);

        // 数字翻转：
        //   将输入的一个整数 翻转  12345 54321
        int y = 123456;
        int d = 0;

        while(y>0){
            int l = y%10;//l=6
            d = d*10+l;

            y/=10;// 整数相除 999/10=99
        }
        // 字符串翻转： abc 12345abc
        String str = "abc";
        // 利用字符串遍历取字符函数  长度： 字符的个数  取字符的下标从0开始 最后一个字符的下标是 length-1
        for (int i = str.length(); i>0 ; i--) {
            char ch = str.charAt(i-1);
            System.out.print(ch);
        }
    }


}



```


- 将数字转成字符串，逐个字符比较

```java

package oop.zyf1224;

public class Main3 {
    public static void main(String[] args) {
        int num = 1232322;
        String numStr = num+"";
        int len=numStr.length();
        for (int i = 0; i <=numStr.length()/2; i++) {
            if(numStr.charAt(i)!=numStr.charAt(len-i-1)){
                System.out.println("不是回文数");
                break;
            }
        }

    }
}
```

## 优化时间复杂度 改成先判断是否是质数 再判断是否是回文数 


```java
package oop.zyf1224;

import java.util.Scanner;

public class Main2 {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        int count = 0;
        for (int i = a; i < b; i++) {
            boolean isPri = true;
            for (int j = 2; j <= i / 2; j++) {
                if (i % j == 0) {
                    isPri = false;
                    break;// 跳出循环
                }
            }
            if (isPri) {
                // 判断是否是回文数
                int n = i;
                int y = n;
                int d = 0;

                while (y > 0) {// 10位 循环10次   换一个循环 从1位开始 与 最后一位比较 只需要循环5次
                    int l = y % 10;//l=6
                    d = d * 10 + l;

                    y /= 10;// 整数相除 999/10=99
                }
                if (d == n) {
                    System.out.println(n);
                    count++;
                }
            }

        }
//        System.out.println(count);


    }

}
```
