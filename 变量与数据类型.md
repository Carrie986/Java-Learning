# 变量与数据类型

## 什么是变量？

- 内存中的一块存储空间，用于存储程序运行过程中可能变化的数据。
### 变量的三要素：
- 数据类型：决定变量存储什么类型的数据。
- 变量名：变量的名字，访问这块内存空间时用的标识。
- 变量值：存储的数据。
-  格式：数据类型 变量名 = 变量值；
```java
// 先声明，再换行赋值
int age;
age = 18;
// 声明的同时赋值
int age = 18;
// 一行声明多个同类型变量
int x = 1, y = 2, z = 3;
```
### 变量命名规则：
- 由字母（大小写），数字，下划线`_`，和美元符号`$`组成。
- 不能以数字开头！
- 大小写有区别：`name`和`Name`是不同变量。
- 不能完全等于Java关键字：如public，class，int，但可以作为变量名的一部分：如`classAverage`。
- 驼峰命名法：如`studentName`，`studentAge`。
## 八大基本数据类型（Primitive Types）
| 数据类型     | 关键字  | 字节 | 取值范围                | 默认值 |
| ------------ | ------- | ---- | ----------------------- | ------ |
| 字节型       | byte    | 1    | -128 ~ 127              | 0      |
| 短整型       | short   | 2    | -32768 ~ 32767          | 0      |
| 整型         | int     | 4    | -2^31 ~2^31-1（±21亿）  | 0      |
| 长整型       | long    | 8    | -2^63 ~ 2^63-1          | 0L     |
| 单精度浮点型 | float   | 4    | ±3.4E-38 ~ ±3.4E+38     | 0.0f   |
| 双精度浮点型 | double  | 8    | ±1.7E-308 ~ ±1.7E+308   | 0.0d   |
| 字符型       | char    | 2    | 0 ~ 65535 (Unicode字符) | ''     |
| 布尔型       | boolean | 1    | true/false              | false  |
- 代码示例：
```java
public class PrimitiveTypes{
	public static void main(String[] args){
		//整数型：
		//byte
		byte byte1 = 100; //范围：-128-127
		System.out.println("byte类型的值："+ byte1);
		
		//short
		short short1 = -32000; //范围：-32768-32767
		System.out.println("short类型的值："+ short1);
		
		//int
		int int1 = 2100000000; //范围：约正负21亿
		System.out.println("int类型的值："+ int1);
		
		//long
		long long1 = 9000000000000L; //注意：数字后面要加'L'!
		System.out.println("long类型的值："+ long1);
		
		// 浮点型：
		//float：约6-7位有效数字*
		float float1 = 3.1415; //注意：数字后面要加'f'或'F'!
		System.out.println("float类型的值："+ float1);
		
		//double：约15-16位有效数字
		double double1 = 3.14159265358; 
		System.out.println("double类型的值："+ double1);
		
		//科学计数法
		double scientific = 2.6E-2; //等于2.6*10^-2，输出值：0.026
		System.out.println("科学计数法的值："+ scientific);
		
		//字符型：
		//char
		char a = 'A'; //输出 A
		char b = 65; //Unicode编码，输出 A
		char c = '\u0041'; //Unicode转义，输出 A
		char d = '好' //输出：好
		System.out.println("char a："+ char a);
		System.out.println("char b："+ char b);
		System.out.println("char c："+ char c);
		System.out.println("char d："+ char d);
		
		//常见转义字符：
		// 常见转义字符
		System.out.println("换行符: Hello\nWorld"); // \n 换行
		System.out.println("制表符: Hello\tWorld"); // \t 制表符，相当于tab键
		System.out.println("双引号: \"Hello\""); // \" 双引号，输出 "Hello"
		System.out.println("反斜杠: \\Hello"); // \\ 反斜杠，输出\Hello
		
		//布尔类型：
		//boolean
		boolean b1 = true; //只能等于true或者false
		System.out.println("布尔值1："+ b1);
		//用于条件判断：
		boolean isAdult = true;
		if (isAdult){
			System.out.println("用户已成年");
		}
	}
}
```
### 一些需要注意的：
- 有效数字：小数点不算一位，只计数实际有效数字。
	- 例子：3.14—3位有效数字；0.0103—3位有效数字（1，0，3）；9.0210—5位有效数字（最后一个0也是有效数字！增加了精确度）。
- Unicode：给每一个字符分配一个唯一编号。
	- 寻找一个内容的Unicode编号的方法：
	```java
	char ch = 'A';
	int num = ch;
	System.out.println(num);
	```
- 单引号和双引号：
	- 双引号 " 用来表示字符串 String，单引号 ' 用来表示单个字符 char。
	- char 只能放一个字符，比如 `char ab = 'AB' `就是错的。
	- 例子：
```java
int a = 5;
	char b = '5';
	string c = "5";
	System.out.println(5+1); //输出6
	System.out.println('5'+1); //输出54
	System.out.println("5"+1); //输出51
```
- 特殊字符：
```java
char newline = '\n'; // 换行
char tab = '\t'; // 制表符
char quote = '\''; // 单引号字符
char slash = '\\'; // 反斜杠字符
```
## 引用数据类型
- 除了八大基本数据类型，其它的都是引用数据类型，如`String`。
### 常见类型：
- String 字符串
```java
// String
String str1 = "Hello";
String str2 = new String("World");
String str3 = str1+""+str2; // 字符串拼接
System.out.println(str3); // 输出 Hello World
```
- array 数组
	- 注意：数组下标（index）从 0 开始，不是从 1 开始。
- null 空值
	- 表示没有指向任何对象，是引用类型的默认值。
```java
// array
int[] numbers = {1,2,3,4,5};
System.out.println("数组长度="+ numbers.length);
// 输出数组长度=5
System.out.println(numbers[0]); // 输出1
// index:    0  1  2  3  4
// value:    1  2  3  4  5

//null：这个引用类型变量没有指向任何对象。
//以下两个不一样：
String a = ""; // 空字符串：指向一个真实存在的字符串对象，但这个字符串里面什么字符都没有。
String b = null; // 空值：没有指向任何对象。
System.out.println(a.length()); // 输出0
System.out.println(b.length());
// 报错：NullPointerException（空指针异常）。
```
## 变量的作用域
```java
public class Test {
// 成员变量（member variable）：在整个类中可用
static int classVar = 10;
public static void main(String[] args) {
// 局部变量（local variable）：只在此方法中可用
int methodVar = 20;
System.out.println("成员变量: " + classVar);
System.out.println("局部变量: " + methodVar);
// 块作用域
{
int blockVar = 30;
System.out.println("块内变量: " + blockVar);
// 访问外部变量
System.out.println("访问局部变量: " + methodVar);
}
System.out.println(blockVar); // 错误：块内变量不能在块外使用
}
```
